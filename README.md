# cs81-module3b-toolkit
CS81 Math Toolkit Functions

--- FUNCTIONS ---  

double(n): Takes in a number n and returns the double  
square(n): Takes in a number n and returns the square  
isEven(n): Takes in a naumber n and returns true is n is even; false otherwise  
abs(n): Takes in a number n and returns the absolute value of n  
factorial(n): Takes in a number n and returns n factorial
